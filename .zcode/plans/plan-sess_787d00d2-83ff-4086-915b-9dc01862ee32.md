## Full Fix Plan

### 1. Hotpic Album Grouping + Direct Video Fix

**scraper.js**: `scrapeHotpic()` returns album objects with `_isAlbum: true` and `_albumVideos[]` array (one entry per video with title, videoUrl, thumbnail).

**core.js**: `handleScrapeAction()` detects `_isAlbum` → sends as **media group** (`replyWithMediaGroup`) instead of individual posts. Each video in group has its own thumbnail + caption with part number.

**Direct video fix**: In `handleScrapeAction()`, wrap `replyWithVideo(url)` in try/catch. On failure → call `downloadVideo()` + `replyWithVideo({source: stream})` fallback. This applies to all sites, not just Hotpic.

---

### 2. Auto-send Menu + Group Topic Routing

**core.js**: Add "⚙️ Auto-Send" button to `getMainMenu()` → opens settings keyboard with:
- Toggle daily digest on/off (`/daily` already exists)
- Site selection (`/settings` already exists)
- **NEW**: "Send to Group Topics" toggle + force channel

**Group topic delivery** (`api/autosend.js` + new `api/groupsend.js`):
- When user has groups with topics, bot scans topics for tag matches (Tamil, Mallu, etc.)
- Posts from each site go to matching topic by tag name
- Trending from each site sent sequentially (not combined AIO)

---

### 3. Admin System + Force Channel

**kv-storage.js**: Add `adminUsers` set (first user = admin). New functions:
- `isAdmin(userId)`, `addAdmin(userId)`, `removeAdmin(userId)`
- `getForceChannel()`, `setForceChannel(channelId)`, `checkForceSubscribe(userId)`

**core.js**: Middleware `forceSubscribeMiddleware` checks membership before any command.
- Admin commands: `/adduser @user`, `/removeuser @user`, `/forcechannel @channel`, `/broadcast msg`

---

### 4. Back-to-Main Fix

**core.js**: `back_to_main` handler - ensure it works for both text and photo messages. The current logic handles photo messages but may fail on video messages. Add check for `ctx.callbackQuery.message.video` and use `replyWithMarkdown` instead of `editMessageText`.

---

### 5. Implementation Order

1. **Hotpic album + video fallback** (scraper.js + core.js)
2. **Auto-send menu + group topics** (core.js + api/autosend.js + new api/groupsend.js)
3. **Admin + force channel** (kv-storage.js + core.js)
4. **Back-to-main fix** (core.js)
